# GET请求所需参数

- URL：
  接口(网址)
- headers：
  告诉服务器你是谁
- token：
  让服务器知道你是否已经登录
  因为有些数据是私有的，不是所有人都能访问
  当你登录成功后，服务器会返回token给你
  token是headers内的内容
- params:
  告诉服务器你需要获取的具体数据

# POST请求所需参数

- URL
- json数据：
  你要提交给服务器的数据
  (用来登录/注册/提交表单/创建数据等)
- data：
  老接口用data来发送表单数据
- hraders：
  告诉服务器你是谁
  1、模拟浏览器
  2、携带token(登录后获取的token)
  3、指定发送给服务器的数据的格式
  "Content-Type":我发送的数据格式
  "Accept":我希望服务器返回的数据格式
